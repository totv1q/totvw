# TOTV+ Premium - دليل التثبيت والاستخدام

## 📱 نظرة عامة
تطبيق TOTV+ هو مشغل IPTV احترافي محسّن لأجهزة iPhone مع دعم كامل لبروتوكول HTTP وتسجيل دخول دائم.

- **السيرفر:** `http://silka.m3um9xplus.org:2052`
- **اسم المستخدم:** `totv5201093051`
- **كلمة المرور:** `17348547711072`

### ✅ كيفية الدخول:
1. افتح التطبيق
2. أدخل الرمز: **1212**
3. اضغط "دخول"
4. سيتصل التطبيق بالسيرفر تلقائياً

**ملاحظة مهمة:** لن يُطلب منك تسجيل الدخول مرة أخرى، حيث يتم حفظ الجلسة تلقائياً!

## 📲 تثبيت التطبيق على iPhone

### الطريقة الأولى: إضافة للشاشة الرئيسية (PWA)

1. **افتح التطبيق في Safari**
   - افتح ملف `totv-plus-fixed.html` في متصفح Safari

2. **اضغط على زر المشاركة**
   - اضغط على أيقونة المشاركة (المربع مع السهم لأعلى) في الأسفل

3. **اختر "إضافة إلى الشاشة الرئيسية"**
   - مرر لأسفل واختر "Add to Home Screen"

4. **اضغط "إضافة"**
   - اضغط "Add" في الزاوية العلوية

5. **الآن التطبيق على شاشتك الرئيسية!**
   - ستجد أيقونة TOTV+ على الشاشة الرئيسية
   - افتحه كأي تطبيق عادي

## 🌐 حل مشكلة HTTP على iPhone

### لماذا يحتاج التطبيق HTTP؟
السيرفر يستخدم بروتوكول HTTP (غير المشفر)، ولكن iOS يفضل HTTPS المشفر.

### ✅ الحلول المُطبقة في التطبيق:

#### 1. **Content Security Policy**
تم إضافة هذا الكود في رأس الصفحة:
```html
<meta http-equiv="Content-Security-Policy" content="upgrade-insecure-requests">
```
هذا يسمح للتطبيق بالعمل مع HTTP.

#### 2. **ملف Manifest.json**
يحتوي على إعدادات PWA التي تسمح بتشغيل HTTP:
- `"display": "standalone"` - يجعل التطبيق يعمل كتطبيق مستقل
- `"protocol_handlers"` - يدعم بروتوكول HTTP

#### 3. **Safari Configuration**
عند إضافة التطبيق للشاشة الرئيسية، سيعمل في وضع standalone مما يتجاوز بعض قيود HTTP.

## ❓ الأسئلة الشائعة

### س: هل يعمل التطبيق بدون إنترنت؟
**ج:** لا، تحتاج لاتصال إنترنت لتشغيل القنوات والمحتوى.

### س: هل يمكنني تغيير بيانات السيرفر؟
**ج:** نعم، افتح ملف `totv-plus-fixed.html` وعدّل القيم في قسم `CONFIG`:
```javascript
const CONFIG = {
  PIN: '1212',                              // غيّر الرمز هنا
  SERVER: 'http://your-server.com:port',    // غيّر السيرفر هنا
  USERNAME: 'your-username',                 // غيّر اليوزر هنا
  PASSWORD: 'your-password'                  // غيّر الباسورد هنا
};
```

### س: لماذا يطلب مني رمز PIN في كل مرة؟
**ج:** إذا كنت تفتح التطبيق من Safari مباشرة، ستحتاج لإدخال الرمز. ولكن إذا أضفته للشاشة الرئيسية، سيتذكر تسجيل دخولك.

### س: ماذا لو نسيت رمز PIN؟
**ج:** الرمز الافتراضي هو **1212**. إذا غيرته ونسيته، افتح الملف وابحث عن `PIN: '...'` لمعرفة الرمز.

### س: هل يمكن تشغيل القنوات في الخلفية؟
**ج:** نعم، عند إضافة التطبيق للشاشة الرئيسية، يمكنك الاستماع للقنوات في الخلفية.

## 🛠️ حل المشاكل

### مشكلة: التطبيق لا يتصل بالسيرفر
**الحل:**
1. تأكد من اتصالك بالإنترنت
2. تحقق من أن عنوان السيرفر صحيح
3. جرب إعادة تحميل الصفحة

### مشكلة: الفيديو لا يعمل
**الحل:**
1. تأكد من اشتراكك فعّال
2. جرب قناة أخرى
3. أغلق التطبيق وافتحه مرة أخرى

### مشكلة: التطبيق بطيء
**الحل:**
1. امسح الـ Cache من Safari
2. أعد تشغيل جهازك
3. تأكد من سرعة الإنترنت جيدة
